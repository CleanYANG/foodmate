# CityTalk

A clean, beginner-friendly Expo + React Native starter for the CityTalk MVP.

## Stack

- Expo
- React Native
- TypeScript
- React Navigation

## Included

- Simple stack navigation
- Minimal theme tokens
- Placeholder screens
- ESLint + Prettier setup
- Clean starter folder structure

## Screens

- Home
- Place Detail
- Saved Places
- Profile

## Commands

```bash
npm install
npm run start
npm run lint
npm run typecheck
npm run format
```

## Folder structure

```txt
src/
  components/
  navigation/
  screens/
  theme/
```

## Notes

This project is intentionally small and easy to extend. Start by replacing the Home screen placeholder with your swipeable place feed.
